Bem-vindo!

Este tutorial foi criado para ensinar os princípios básicos da lógica de programação para iniciantes. Vamos abordar conceitos essenciais e práticos para você iniciar na programação de forma simples e eficaz.

Conceitos Fundamentais

1. Variáveis

Armazenam valores para uso posterior. Exemplo: nome = "Maria"

2. Tipos de Dados

Números, textos. Exemplo: idade = 18, ativo = True

3. Condicionais

Executam diferentes códigos com base em condições. Exemplo: if idade >= 18:
    print("Maior de idade")

Exercícios Práticos

  1. Crie uma variável com seu nome e imprima-a.
  2. Peça ao usuário um número e diga se é par ou ímpar.
  3. Escreva uma função que receba dois números e retorne a soma.
